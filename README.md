# Ecommerce-SC-Analytics

This project analyzes e-commerce order datasets to extract insights and visualize trends.

## Features
- Data loading and preprocessing
- Exploratory data analysis (EDA)
- Visualization of key metrics

## Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/Ecommerce-SC-Analytics.git
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Usage
Run the `load_data.py` script to download and process the dataset:
```bash
python load_data.py
```
